# Harsh Singh - AI/ML Developer Portfolio 🚀

Welcome to the repository for my personal portfolio website! This project showcases my experience, projects, and skills as an **AI/ML Developer and Business Analyst**.

## 🌟 Features

- **Modern & Interactive UI**: Premium and dark-themed design built with React, TypeScript, and Vanilla CSS.
- **Smooth Animations**: Integrated with GSAP and Lenis for buttery-smooth scrolling and beautiful element reveal effects.
- **Dynamic Content**: Data-driven components managed via `config.ts` for easy updates to projects, experiences, and about me sections.
- **Downloadable Resume**: Built-in functionality to directly download my latest PDF resume.
- **Responsive Layout**: Fully optimized for desktop, tablet, and mobile viewing.

## 🛠️ Tech Stack

- **Framework**: [React](https://react.org/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Custom CSS with modern property variables
- **Animations**: [GSAP](https://gsap.com/) & [Lenis Smooth Scroll](https://github.com/darkroomengineering/lenis)

## 🚀 Getting Started

To run this project locally, simply follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/HaRsH00000007/Portfolio_Website_ver_1.git
   cd Portfolio_Website_ver_1
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to `http://localhost:5173` in your browser to view the portfolio.

## 📂 Project Structure

- `src/components/`: Contains all the raw React components (Navbar, Landing, Work, etc.)
- `src/config.ts`: The central data configuration file where all portfolio text and data are stored.
- `public/`: Contains static assets, generated project images, logos, and the downloadable resume.
